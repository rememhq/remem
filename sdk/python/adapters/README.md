# Framework Adapters

This directory will contain drop-in adapters for popular AI frameworks:

## Planned Adapters

| Framework | Module | Status |
|---|---|---|
| LangChain | `remem.adapters.langchain` | 🚧 v0.2 |
| LlamaIndex | `remem.adapters.llamaindex` | 🚧 v0.2 |
| AutoGen | `remem.adapters.autogen` | 🚧 v0.2 |
| CrewAI | `remem.adapters.crewai` | 🚧 v0.2 |

Install with extras:
```bash
pip install remem[langchain]
pip install remem[autogen]
```
